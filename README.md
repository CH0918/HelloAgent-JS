# HelloAgent-JS

JavaScript version of the Hello Agents LLM client.

## Setup

```bash
npm install
cp .env.example .env
```

Then fill in `.env`:

```env
LLM_MODEL_ID=your-model-id
LLM_API_KEY=your-api-key
LLM_BASE_URL=https://your-openai-compatible-endpoint/v1
LLM_TIMEOUT=60
```

## Run

```bash
npm start
```
